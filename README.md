# THL_ONE_Demo
## Backend
### Chỉnh sửa thông tin file cấu hình database
``` appsetting.json ```
### Tạo database (chạy lệnh trên nuget package console)
``` update-database ```
### Khởi chạy project
``` dotnet run ```
### Document API
```https://localhost:5001/swagger/index.html```
## frontend
### Khởi tạo môi trường
- cài nodejs
- cài angular 12
- ``` npm install @angular/cli -g ```
### Các lệnh để khởi chạy
- ``` cd demo_user\ClientApp ```
- ``` ng build ```
- ``` ng serve ```
